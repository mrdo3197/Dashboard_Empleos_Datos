# Dashboard de Empleos de Datos en Power Bi

## Introducción

Este dashboard fue credo para **personas que están en el proceso búsqueda de trabajo o transicionar hacia un trabajo relacionado con el mundo de los datos** de manera que este tipo de personas puedan encontrar y analizar información del mercado laboral de datos en un lugar centralizado. Usando una *base de datos de empleos relacionados con datos reales para el año 2024* (incluyendo cargos de empleos, salarios y ubicaciones) este proyecto proporciona una interfaz única y fácil de usar para explorar tendencias en el mercado.

## Habilidades

- **Transformación de datos (ETL) con Power Query:** Limpié y preparé los datos crudos para el análisis removiendo datos vacíos, asignando tipos de datos y creando nuevas columnas.

- **Medidas Implícitas**: Creé medidas para derivar hallazgos principales y KPIs como `Mediana del Salario Anual` y `Cantidad de Empleos`.

- **Gráficas Principales**: Utilicé gráficos de **columna, barras, líneas y dispersión** para comparar cantidad de empleos y visualizar tendencias en el tiempo.

- **Análisis Geospacial**: Implementé un **gráfico de mapa** para visualizar la distribución global de los empleos.

- **Indicadores de KPI y Tablas**: Usé **Tarjetas** para mostrar métricas clave y **Tablas** para visualizar y organizar información más detallada.

- **Diseño de Dashboard**: Diseñé una estructura intuitiva y visualmente llamativa explorando tanto tipos de gráficos comunes como no comunes para contar la historia que brindan los datos de la mejor manera.

- **Reporte Interactivo**:

    - **Slicers**: Para filtrar dinámicamente los reportes por Tipo de Empleo.
    - **Botones**: Para crear una experiencia de navegación más sencilla e intuitiva para el usuario.
    - **Drill-Through**: Para navegar de un alto nivel a un nivel detalle más específico de la información presentada.


# Resumen del Dashboard

### Página 1: Vista de alto nivel del Mercado de Empleos de Datos

![Dashboard - Página 1] (\Imágenes\Parte1.png)

Esta es la página principal para la información del mercado laboral de datos. Muestra KPIs clave como cantidad de empleos totales, mediana de salarios y empleos más requeridos para dar una mirada rápida a lo que está sucediendo en el mercado.

### Página 2: Empleos Drill Through

![Dashboard - Página 2] (\Imágenes\Parte2.png)

Esta es la página de detalles más específicos. Desde la página principal del dashboard es posible navegar a esta página para obtener detalles específicos para un único tipo de empleo incluyendo información como rnagos de salario, estadísticas de modalidad del empleo (en este caso si el empleo es remoto o no), las plataformas con más ofertas de empleo y un mapa global de las ubicaciones de dichos empleos.

## Conclusión

Este dashboard expone como Power BI puede transformar datos crudos de vacantes de empleo en una herramienta poderosa para realizar análisis permitiendo al usuario segmentar, filtrar e ir más en detalle para tomar decisiones informadas basadas en datos sobre su carrera profesional.
